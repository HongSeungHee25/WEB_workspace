JSON 형식 특징

  1. 키-값 쌍(Key-Value Pair) : JSON은 이름과 값이 쌍으로 이루어진 객체 형태를 가집니다.
    이름(키)과 값(값)은 콜론(:)으로 구분하며, 쌍들은 중괄호({})로 묶어 표현합니다.
  2. 배열(Array) : JSON은 값들의 순서화된 목록을 배열로 표현할 수 있습니다.
    배열은 대괄호([])로 묶어 표현하며, 값들은 쉼표(,)로 구분합니다.
  3. 데이터 형식 : JSON은 텍스트로 이루어진 데이터 형식입니다.
    기본적으로 숫자, 문자열, 불리언, null, 객체, 배열 등의 데이터 형식을 지원합니다.
  4. 중첩 가능 : JSON은 객체나 배열 안에 다른 객체나 배열을 중첩하여 사용할 수 있습니다.
    이를 통해 복잡한 데이터 구조를 표현할 수 있습니다.
