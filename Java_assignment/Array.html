배열(Array)

배열이란 선형 자료구조(Data Strucyure)중 하나로 동일한 타입의 연관된 데이터를 메모리에 연속적으로 저장하여 하나의 변수에 묶어서 관리하기 위한 자료 구조입니다. 가장 기본적인 자료구조인 만큼 C,Java,Python 등 거의 모든 언어에 구현되어 있습니다. 또한 자바에서 배열은 0번부터 시작해서 진행됩니다.

배열의 사용 목적

  • 규칙이 없는 값에 규칙성을 부여하기 위해
  • 저장 공간 여러 개를 이름 한개로 관리할수 있음

배열의 선언과 생성

  • [타입[] 변수이름 ;] 의 형식과 [타입 변수이름 [];]의 형식으로 선언 가능

" "* 배열을 선언하는 것은 배열을 다루기 위한 참조변수를 위한 공간만 만들어지는 것이고 정말 값이 저장되는 공간을 만들어주기 위해서는 선언한 배열을 생성해주어야 한다.*

  • (배열변수) = new (타입) [(길이)]

배열의 길이와 인덱스

배열의 값을 다루기 위해선 인덱스(index)를 사용한다.

배열의 초기화와 출력

배열 생성 시 배열의 값은 배열의 타입에 해당하는 기본 값으로 초기화

  • 정수형 배열의 초기값 : 0
  • 문자형 배열의 초기값 : ' '
  • 실수형 배열의 초기값 : 0.0
  • 문자열 배열의 초기값 : Null
